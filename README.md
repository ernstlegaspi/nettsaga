# Kristiansand Hundefrisør Redesign

Single-page React + Tailwind redesign of `kristiansandhund.no`, built as a submission-ready assessment piece.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Short note

- Design decision: kept the original site copy for pricing, services, reviews, employee details, and contact labels so the redesign stays faithful to the existing content baseline.
- Design decision: used a warmer editorial direction with `Fraunces` for headings, `Manrope` for body text, and a cream / pine / coral palette to make the page feel more premium and welcoming.
- Design decision: organized the page into clear sections for hero, about, pricing, services, reviews, and contact so users can scan the content faster.
- Improvement: rebuilt the price list into comparison-style cards to make package differences easier to understand at a glance.
- Improvement: strengthened CTA placement by surfacing contact actions early in the hero and again in the contact section.
- Improvement: improved responsiveness with stacked cards, larger tap targets, and cleaner spacing for mobile and tablet layouts.
- Improvement: separated the code into reusable React components and centralized the content in a shared data file to keep the project easier to maintain.
- AI tools used: Codex was used for layout ideation, component structuring, content mapping from the original page, and implementation acceleration inside the React + Tailwind setup.
