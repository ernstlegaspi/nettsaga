# Kristiansand Hundefrisør Redesign

Single-page React + Tailwind redesign of `kristiansandhund.no`, built as a submission-ready assessment piece.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Design notes

- Kept the original site copy for pricing, services, reviews, contact labels, employee details, and footer references.
- Reorganized the page into clearer sections with stronger hierarchy and more deliberate CTA placement.
- Rebuilt the price list as scannable comparison cards instead of a single text block.
- Added a warmer editorial visual style using `Fraunces` + `Manrope`, custom colors, layered gradients, and rounded glassmorphism surfaces.
- Improved mobile readability with stacked cards, larger tap targets, and simplified navigation.
- Used React with Tailwind CSS as the primary styling system.
- AI assistance: layout ideation, content structuring, and component generation were accelerated with Codex.
